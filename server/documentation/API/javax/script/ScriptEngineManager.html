public class ScriptEngineManager
extends java.lang.Object
ScriptEngineManager implements a discovery and instantiation mechanisams for ScriptEngine class. It also contains a collection of key-value pairs which storing state shared by all engines created by manager. Nandika Jayawardana <nandika@opensource.lk> Sanka Samaranayake <sanka@opensource.lk>
