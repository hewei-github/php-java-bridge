Interface ScriptEngineFactory

public interface ScriptEngineFactory
ScriptEngineFactory is used to describe a ScriptEngine instance. Each ScriptEngine class implementing ScriptEngine has a corresponding factory which exposes metadata describing the engine class. Nandika Jayawardana <nandika@opensource.lk> Sanka Samaranayake <sanka@opensource.lk>
