public class ContextServer
extends java.lang.Object
implements java.lang.Runnable
This class manages the ContextRunners. It pulls a ContextRunner from the list of available runners and invokes it. We currently check only if the number of runners requested is less than or equal to the number of runners created by the servlet requests.
