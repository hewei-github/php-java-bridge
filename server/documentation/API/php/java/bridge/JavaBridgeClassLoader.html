Class JavaBridgeClassLoader

  • java.lang.Object
    • php.java.bridge.JavaBridgeClassLoader
public class JavaBridgeClassLoader
extends java.lang.Object
A bridge pattern which allows us to vary the class loader as run-time. The decision is based on whether we are allowed to use a dynamic classloader or not (loader==null) or security exception at run-time.
