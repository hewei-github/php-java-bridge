public class HttpProxy
extends CGIRunner
Represents the script continuation. This class can be used to allocate php scripts on a HTTP server. Although this class accidently inherits from CGIRunner it doesn't necessarily run CGI binaries. If you pass a URLReader, it calls its read method which opens a URLConnection to the remote server and holds the allocated remote script instance hostage until release is called.
