Class URLReader

  • java.lang.Object
    • java.io.Reader
      • php.java.script.URLReader
public class URLReader
extends java.io.Reader
This class can be used to connect to a HTTP server to allocate and to invoke php scripts. Example:

PhpScriptEngine e = new PhpScriptEngine();
e.eval(new URLConnection(new URL("http://localhost:80/foo.php"));
System.out.println(((Invocable)e).call("java_get_server_name", new Object[]{}));
e.release();
